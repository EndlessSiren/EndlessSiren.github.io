Home HTML

HTML pages are text documents. HTMl uses tags to give the text in them extra meaning. Tags are referred to as elements. Tags usually come in pairs, an opening and closing tag. This isn't always the case, some tags are self closing. Opening tags can have attributes that provide futher information on the content of the element. Attributes require a name and a value.

Semantic Markup.

The following elements are not intended to affect the structure of a page but do add extra information.

Strong & Emphasis.

"Strong" indicates that content has strong importance. By default the browser will display it in bold.

Emphasis ("em") idicates ephasis that subtley changes the meaning of a statement. By default browsers will display this in italics.

Quotations.

"Blockquote" is used to quotate an entire paragraph. P tags are still used inside. Browsers usually indent the blockquote, if you want to indent only use CSS.

The "q" element is used to quote shorter text within a paragraph. Intenet explorer does not apply this element though so many people tend to avoid this element.

Abbreviations & Acronyms.

"Abbr" is an element used for abbreviations. Use a title attribute to specify the full term. E.g. LOL

Citations & Definitions.

To cite a source use the "cite" tag. This will render in italics.
The furst time you define a new termminology you can use a dfn element. Some browsers display this in italics but non Safari or Chrome.

Summary

HTML elements are used to define the structure of a page. They can also provide semantic info such as ephasis or definitons.

Lists

Ordered Lists

  1. Ordered lists are created with th "ol" tag.
  2. Each entry on the list is added with "li" tags.
  3. Browsers will indent lists by default.
  4. You can use a type atrribute to select the type like numbers, letters or roman numerals.
  5. It is best practice to use the CSS lists-style-type property.

Unordered Lists

  • Unordered lists are created with the "ul" tag.
  • Again you can use a type attribute but it is best practice to use CSS.
